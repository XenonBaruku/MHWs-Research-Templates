# MHWs-Research-Templates #
010 Editor binary template that parses some files from Monster Hunter Wilds. <br/>
(Currently WIP - Not sure everything is correct)

## Files ##
 * **MHWs_gpuc.bt** – GPU Cloth (*.gpuc) (Not completed yet! Credit [alphaZomega](https://github.com/alphazolam) for [RE8_GPUC.bt](https://github.com/alphazolam/RE-Engine-010-Templates/blob/main/RE8_GPUC.bt) )
 * **MHWs_sfur.bt** – ShellFur (*.sfur)
 * **RE_engine_jcns.bt** – Joint Constraints (*.jcns)  (Supported games: RE2R, DMC5, RE3R, RE8, MHRS, RE4R, SF6, MHWs, PRAGMATA DEMO)
